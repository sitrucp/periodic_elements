This cool periodic chart of the elements shows source / origin of the elements. Source: Wikipedia created by Cmglee

Some elements may come from more than one source which are listed below:

  • Big Bang fusion
  • Exploding white dwarfs
  • Exploding massive stars
  • Cosmic ray fission
  • Merging neutron stars
  • Dying low-mass stars
  • Human synthesis

I wanted to see counts of elements by origin. It turned out that the SVG file contains the data and Python code necessary to create the SVG file itself!

With some minor modification the data could be extracted into a csv data file and used for the visualizations of counts of elements by origin below. (Read more about the modifications here https://github.com/sitrucp/periodic_elements)
